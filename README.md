# Shader - Ojo Animado

## Descripción
Este shader es un efecto visual en GLSL que representa un **ojo animado** sobre un fondo dinámico.  
El ojo está diseñado con forma vertical elíptica y presenta un **gradiente radial de colores** que va del naranja al rojo.  
El fondo incluye ondas radiales que se mueven con el tiempo, dando un efecto de energía o pulso alrededor del ojo.
Como inspiración se tiene el "Ojo de Sauron" del Señor de los anillos y algunos de los ejemplos del github de prácticas. 

---

## Características
- Gradiente radial dentro del ojo: centro naranja → borde rojo.
- Ojo vertical con forma biconvexa.
- Animación suave que simula un **latido** del ojo.
- Contorno definido para mantener la opacidad y claridad.
- Fondo con **ondas radiales animadas**.
- Ligero y optimizado para funcionar en GLSL / WebGL.
